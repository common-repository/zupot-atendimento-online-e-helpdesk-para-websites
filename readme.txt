== Description ==

Zupot &eacute; um software de atendimento online gratuito com suporte para iPhone, Android, Web e tablets. Para iniciar crie sua conta gratuita em www.zupot.com/comece-agora.

== Installation ==

1. Fa&ccedil;a o upload do arquivo `wordpress-zupot` no diret&oacute;rio de plugins (`/wp-content/plugins/`)
2. Ative o login por meio do menu 'Plugins' no WordPress
3. Clique em 'Zupot' no menu de configura&ccedil;&otilde;es &agrave; esquerda
4. Siga os passos.
